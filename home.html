Encore Bridge

Using Encore UI with Design System styling

Goals

This project aims to enable developers to use Encore components in customer-facing applications with minimal changes to the markup. Since not all of the design patterns prescribed in Encore have a Design System equivalent, not every pattern or component has been migrated.

This project does not intend to provide a single flag that can be flipped to turn an internal app into an external one. Development must still target one of those platforms, but now the toolboxes for doing so are more similar.
