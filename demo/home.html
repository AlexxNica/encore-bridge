Encore Bridge

Using Encore UI with Canon styling

Goals

This project aims to enable developers to use Encore components in Canon-styled applications with minimal changes to the markup. Since not all of the design patterns prescribed in Encore have a Canon equivalent, not every pattern or component has been migrated.

This project does not intent to provide a single flag that can be flipped to turn an Encore app into a Canon one. Development must still target one of those platforms, but now the toolboxes for doing so are more similar.
